## What is this branch?

This branch is archived because the Graphite core was completely rewritten at the time this branch was created (and the pre-rewrite version is maintained here). This is a legacy version, meaning it has no support.

## Branch Content

This branch contains version `0.1-dev1` of Graphite before the structural changes were applied, it was converted to a standard module, and *Core Rewrite #1*. This version lacks fundamental features and may have major security flaws. **_Never_** use this version in a test or live environment, the only reasonable use of this version is to study the code without running it.

## More Information

This version is a completely experimental (sandbox) implementation of Graphite, its DSL version is compatible with Graphite 0.2 according to the latest review, although there may be differences in behavior or missing features. It contains some basic queries, a number of queries by name only, structure and data parsing functions, overview retrieval functions, node and relationship creation functions, and limited validation functions.
